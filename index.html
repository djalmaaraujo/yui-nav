YIU3 Nav Widget

This is a YUI3 Widget created for experimental studies on YUI. The widget have some cool features as you can see below:

  • Accessible Navigation;
  • Orientation param [Vertical or Horizontal];
  • Navigation can be built from a HTML Markup or a JSON object with some cool stuff;
  • On JSON parser, you can add some click: {} event handler;
  • Load multiple CSS to the nav;
  • Re-usable widget with other YUI instances. Just use as a module;

USING WIDGET

CALL YUI

<script src="http://yui.yahooapis.com/3.4.1/build/yui/yui-min.js"></script>

CALL NAV WIDGET

<script src="/your_path/yui3-nav-widget.js"></script>

JSON OBJECT

YUI().use('nav-widget', function (Y) {

    var navigation = new Y.NavWidget({
      srcNode: '#load-menu-here', // HTML element to load the NAV
      orientation: 'vertical', // orientation param (optional)
      dataSource: [ // Your source
        {
        content: '<a href="/">Home</a>',
        },
        {
        content: '<a href="/">About</a>',
        },
        {
        content: '<p>Services</p>', // This can be any HTML
        },
        {
        content: '<a href="/">Categories</a>',
        childrens: [ // You can create infinit nested childrens
            {
            content: '<a href="/">Category I</a>',
            },
            {
            content: '<a href="/">Category II</a>',
            click: function (e) { // You can pass a click event
                alert('Alert this and continue to link!');
                e.stopPropagation();
              },
            },
            {
            content: '<a href="/">Category III</a>',
            }
          ]
        }
      ]
    });
    navigation.render();
});
